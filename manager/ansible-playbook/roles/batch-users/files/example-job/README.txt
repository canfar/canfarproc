
The file canfar-test.sub contains an example job submition file. To submit it type:

canfar_submit canfar-example.sub

You can optionally include the VM image to use in the submit file or as a command line argument, like this:

canfar_submit canfar-test.sub canfar-rocky-8

The VM image specified will be transfered to the batch cloud(s) automatically from your personal arbutus project if it does not already exist there.
